# Document RAG App

## Introduction

This simple Streamlit application allows users to upload their document in a pdf, or txt format. And then ask question about these documents in a chatbots format.

The chatbot interface allows users to choose certain tags when asking questions. If I have a few documents about Low - Rank Adaptation models I could click the "Machine Learning" tag to narrow the scope of the discussion. Preventing any generalized statements that may appear from other documents. 

In the left hand menu there is the option to add folders, tags, and documents. 
When uploading documents the user is prompted to choose the folder location and tags, allowing for control of scope later.
